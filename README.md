# Sistema de Hospedagem

Projeto simples desenvolvido para praticar os conceitos de Programação Orientada a Objetos em C#.  
A proposta é simular um sistema de reserva de hospedagens, utilizando classes como `Pessoa`, `Suite` e `Reserva`.

## Funcionalidades

- Cadastro de hóspedes
- Definição de suíte com tipo, capacidade e valor da diária
- Reserva com cálculo do valor total
- Desconto de 10% para reservas acima de 10 dias
- Validação da capacidade da suíte

## Como executar

1. Clone o repositório ou baixe os arquivos.
2. No terminal, navegue até a pasta do projeto.
3. Execute os comandos:

```bash
dotnet build
dotnet run
